UI Pattern Library

Navigation Bar

Example

  • Products
  • Brands
  • Vacuum Finder
  • Help
  • FAQs

Description

Our main navigation module will be static and serving as our header. Our products and brands buttons will be on-hover drop down menus that serves as the filter categories.

Code

HTML

<nav class="main-nav">
    <div class="left">
         <ul>
            <li><a href="">
                <img class="logo" src="assets/img/logo.png" alt="CleanUp Logo">
             </a></li>
        </ul>
    </div>
    <div class="center">
        <ul>
            <li><a href="">Products</a></li>
            <li><a href="">Brands</a></li>
            <li><a href="">Vacuum Finder</a></li>
            <li><a href="">Help</a></li>
            <li><a href="">FAQs</a></li>
        </ul>
    </div>
    <div class="right">
        <ul>
            <li><a href=""><ion-icon name="search-outline"></ion-icon></a></li>
            <li><a href=""><ion-icon name="person-circle-outline"></ion-icon></a></li>
            <li><a href=""><ion-icon name="cart-outline"></ion-icon></a></li>
        </ul>
    </div>
</nav>

CSS

nav.main-nav {
    background: #F5F9FF;
    padding: 1rem;
    border: 1px solid #ddd;
    display: grid;
    grid-template-columns: 2fr 8fr 2fr;
}

nav.main-nav .center {
    text-align: center;
}                                   
                                        
nav.main-nav .right {
    text-align: right;
    font-size: 1.25rem;
}                                  
                                        
nav.main-nav ul {
    margin: 0;
    padding: 0;
    height: 100%;
}                                     

nav.main-nav ul li {
    display: inline-block;
    height: 100%;
    padding: 0.75rem 1rem 0 1rem;
}                                      

nav.main-nav img.logo {
    margin-top: -0.1rem;
}

Your Review

Would you recommend this product to others?

Please give your experience a rating

Code

HTML

<div class="review-form">
<h2>Your Review</h2>
<form action="">
<div class="input-group">
<label for="review-title">Review title</label>
<input type="text" id="review-title" placeholder="Review Title">
</div>
<div class="input-group">
<label for="review-title">Review body</label>
<textarea id="review-title" placeholder="Your review"></textarea>
</div>
<div class="review-form-half">
<div class="input-group">
<p>Would you recommend this product to others?</p>
<input type="radio" id="yes" name="recommend" value="yes">
<label for="yes">Yes</label>
<input type="radio" id="no" name="recommend" value="no">
<label for="no">No</label>
</div>
<div class="input-group">
<p>Please give your experience a rating</p>
<div class="rating-cluster">
<ion-icon name="star-outline"></ion-icon>

<ion-icon name="star-outline"></ion-icon>
<ion-icon name="star-outline"></ion-icon>
<ion-icon name="star-outline"></ion-icon>
</div>
</div>
</div>
<input type="submit">
</form>
</div>
