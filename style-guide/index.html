Basic Styles

Headings

Example

This is Heading 1

This is Heading 2

This is Heading 3

Description

Heading 1 is ssed for large organism headers, most pages will only have one. Heading 2 is used specifically for body paragraph headers; sometimes with just a few other elements within the same block as the header. Used for table headers, directory paths, and for any minor headers.

Code

HTML

    <h1>This is Heading 1</h1>
    <h2>This is Heading 2</h2>
    <h3>This is Heading 3</h3>

CSS

    h1, h2, h3 {
        font-family: aktiv-grotesk, sans-serif; 
    }

Description

The in-text link is used for less encouraged functionalities that goes against the ecommerce goal of selling products. Most of these will be seen on our checkout pages with examples such as "remove item".

Code

HTML

<p>Lorem ipsum dolor sit amet <a href="">consectetur</a> adipisicing elit.</p>

    a {
        color: #69e;
        text-decoration: none;
    }

    a:hover {
        color: #4D80D5;
    }

Call To Action Buttons

Example

CTA Button Ghost Button

Description

This call to action is designed to utilize color to denote a functionality allowing user to recognize interactive elements easier. The slightly muted blue also creates less of an agressive need for the interaction as we want the user experience to be relaxed and explorative. The ghost buttons are for buttons next to CTA buttons with less density in design resulting in less of an attraction for interaction (secondary CTA buttons).

Code

HTML

<a href="" class="button"> CTA Button </a>
<a href="" class="button ghost"> Ghost Button </a>

CSS

    a.button {
        padding: 0.75rem 1rem;
        background: #69e;
        border: 2px solid #69e;
        color: #fff;
        text-decoration: none;
        border-radius: 4px;
        transition: all 0.3s ease;
        text-align: center;
        font-size: 1rem;
    }

    a.button.ghost {
        background: transparent;
        color: #69e;
    }

    a.button:hover, a.button:focus {
        box-shadow: 0 2px 10px 1px rgb(160,198,255);
        background: #4D80D5;
        border-color: #4D80D5;
    }

    a.button.ghost:hover, a.button.ghost:focus {
        color: #fff;
        background: #69e;
        border-color: #69e;
    }

Submit Buttons

Example

Description

Submit buttons will be the same as call to action buttons for the same reason outlined. The CSS and javascript will carry out the intended functionality.

Code

HTML

<form action="">
<input type="submit" value="Submit Button">
</form>

CSS

    input[type="submit"] {
        background: #69e;
        color: #fff;
        border-color: #69e;
        cursor: pointer;
    }

    input[type="submit"]:hover, input[type="submit"]:focus {
        box-shadow: 0 2px 10px 1px rgb(160,198,255);
        background: #4D80D5;
        border-color: #4D80D5;
    }

Inputs

Example

Description

Text input and labels will be stacked on top of each other for easier design balance. The text input boxes will turn from grey to our primary blue when focused (clicked) on for responsiveness.

Code

HTML

<label for="input-demo">Label</label>
<input type="text" id="input-demo" placeholder="Placeholder Text">

CSS

    input, textarea {
        border-radius: 4px;
        box-shadow: none;
        border: 2px solid #ddd;
        padding: 0.5rem 1rem;
        transition: all 0.3s ease;
        font-size: 1em;
        display: block;
        font-family: aktiv-grotesk, BlinkMacSystemFont, -apple-system, system-ui, "Segoe UI", Roboto, "Helvetica Neue", sans-serif;
    }

    input:focus, textarea:focus {
        outline: none;
        border: 2px solid #69e;
    }

    input[type="text"], textarea {
        padding: 0.5rem 0.75rem;
    }

    label {
        margin-bottom: 0.25rem;
    }
